Tip menghindari banned WhatsApp
  • Gunakan random delay antar pesan (default 8-25 detik, manusiawi).
  • Aktifkan batch break untuk volume besar — istirahat 2-5 menit tiap 30-50 pesan.
  • Hindari kirim ke nomor yang belum pernah chat. Banyak "unknown number" = trigger antispam WA.
  • Variasikan isi pesan dengan spintax: {Halo|Hai|Hi} akan dirandom per pesan.
  • Batas aman tidak resmi: ≤200 pesan/hari untuk nomor baru, ≤500/hari untuk nomor lama.
